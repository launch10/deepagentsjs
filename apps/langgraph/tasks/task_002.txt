# Details:
1. Identify where template files are currently loaded into the WebContainer
2. Prepare template files in the FileSystemTree format:
```typescript
const files: FileSystemTree = {
  'index.js': {
    file: {
      contents: '// Your code here'
    }
  },
  'public': {
    directory: {
      'index.html': {
        file: {
          contents: '<html>...</html>'
        }
      }
    }
  }
};
```
3. Alternatively, consider using @webcontainer/snapshot for binary snapshots if template files are complex
4. Replace current file loading logic with mount():
```typescript

```
5. Ensure this is called early in the page load lifecycle
