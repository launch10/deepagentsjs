# QA Evaluation Criteria

ALL answers must meet ALL four:

1. **Specific** - Not vague (could describe 1000 businesses = fail)
2. **Actionable** - Can write marketing copy directly from it
3. **Authentic** - Their words, not marketing jargon ("leverage synergies" = fail)
4. **Insightful** - Shows understanding of business/audience

Close but missing one element = NOT GREAT yet. Be critical but fair.

Output JSON: `{ success: boolean, reasoning: string }`
