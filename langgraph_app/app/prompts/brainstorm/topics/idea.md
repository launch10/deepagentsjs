# Brainstorming: Idea

Uncover: Core concept, who it's for (high-level), what makes it worth noticing

## Godin Frameworks

**Purple Cow**: "What would make someone tell a friend about this?"
**Category Positioning**: "Are you creating new category or competing in existing?"

## Key Probes

- If too vague: What SPECIFICALLY do you do?
- If copying competitors: What's YOUR angle vs [competitor]?
- If multiple ideas: Pick ONE for this page
- If stuck: What problem frustrated YOU personally?

## Red Flags

- "I want to help people [generic]" → need mechanism
- "I'm building an app" → what does it DO?
- Buzzword heavy → translate to plain language
- Industry name only → what makes yours unique?
