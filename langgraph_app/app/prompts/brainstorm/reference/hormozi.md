# Hormozi Principles - Quick Reference

**Value Equation**: `Value = (Dream Outcome × Likelihood) / (Time × Effort)`

Maximize top (bigger outcome, more credible), minimize bottom (faster, easier).

**Dream Outcome**: Quantify transformation ("add $50K/mo" not "grow business")

**Perceived Likelihood**: Why it works—unique mechanism, track record, guarantee

**Time Delay**: Speed to first win, full transformation timeline

**Effort & Sacrifice**: How hard? Done-for-you > Done-with-you > DIY

**Unique Mechanism**: Name your process. "4-Phase System" not "our approach". Prevents commoditization, enables premium pricing.

**Grand Slam Offer**: Stack value so high price seems stupid to refuse. Core + bonuses + guarantee + scarcity.

**Guarantees**: Strongest = unconditional. Conditional/outcome-based if confident. Removes risk = increases likelihood.

**Pricing**: Charge max you can with straight face, add enough value to make no-brainer. Higher prices attract better customers.

**Value-to-Price Gap**: Show $20K value, charge $3K. Gap makes it feel like stealing.

## When to Apply
- Solution weak → Apply Value Equation (score each variable)
- Generic offering → Add Unique Mechanism
- Underpriced → Show value stack, increase price
- Risk concerns → Add guarantee
- Feature-focused → Convert to outcome-focused
